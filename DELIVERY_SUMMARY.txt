🔄 BOOKING STATE MACHINE

START: pending
  ├─ CONFIRM: pending → confirmed (Host accepts)
  │  ├─ HANDOVER: confirmed → in_use (QR scan by host)
  │  │  ├─ RETURN: in_use → returned (QR scan by guest)
  │  │  │  └─ COMPLETE: returned → completed (Auto/manual)
  │  │  └─ DISPUTE: in_use → disputed (Guest raises issue)
  │  ├─ CANCEL: confirmed → cancelled (Before start)
  │  └─ DISPUTE: confirmed → disputed
  ├─ CANCEL: pending → cancelled
  └─ DISPUTE: pending → disputed

END: completed, cancelled
SPECIAL: disputed → completed (Admin resolves)

💰 COMMISSION BREAKDOWN

Formula:
  Rental Amount = Daily Price × Days
  Commission Host = 12% of Rental Amount
  Commission Guest = 6% of Rental Amount
  Platform Commission = 18% total

Example (₹500/day, 5 days):
  Rental = ₹2500
  Commission Host = ₹300 (12%)
  Commission Guest = ₹150 (6%)
  Host Payout = ₹2200 (₹2500 - ₹300)
  Guest Total = ₹2650 (₹2500 + ₹150)
